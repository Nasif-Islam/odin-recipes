# Odin Recipes

A basic recipe website built for [The Odin Project] (https://www.theodinproject.com/) Foundation Course.

## Live Preview

[View the live site on GitHud Pages] (https://your-username.github.io/odin-recipes/)

## Porject Description

This project is a simple website showcasing a few of my favorite recipes. The main goal was to practice and solidy my understanding of basic HTML, including:
*   Setting up a proper HTML boilerplate.
*   Using essential HTML tags (`<h1>`, `<p>`, `<a>`, `<img>`, `<ul>`, `<ol>`, `<li>`).
*   Creating relative links between multiple pages.
*   Structuring a website with a main index page and linking to sub-pages.
